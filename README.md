# Decibéis — a turnê contra O Empresário

Jogo RPG de turnos com minigame de ritmo, desenvolvido em Python (POO) com frontend HTML/CSS/JS via pywebview.

> README completo (instalação, execução, UML, padrões POO) será adicionado na Fase 4 (Entrega).
